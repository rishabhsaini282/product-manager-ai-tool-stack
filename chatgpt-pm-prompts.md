# Advanced ChatGPT Prompting for Product Managers

Basic prompting yields generic, hallucinated advice[cite: 6]. Elite product leaders use constrained context window engineering to automate strategic analysis securely[cite: 8]. 

## 1. Context Inflation Avoidance
Flooding the prompt with unorganized data degrades the model's reasoning[cite: 8]. 
*   **Rule:** Use distinct Markdown delimiters (`###`, `---`, `>`) to separate business logic, historical performance metrics, and strict technical constraints[cite: 8].
*   **Rule:** Ground prompts with factual, source-of-truth reference data and instruct the model to state when it lacks information to prevent hallucinating technical APIs[cite: 8].

## 2. The Competitor SWOT Framework
Analyze competitors without exposing your own strategy.
*   **Input:** Paste public financial reports, user reviews from app stores, and marketing copy[cite: 8].
*   **Command:** Categorize qualitative user complaints into actionable feature requests and identify functional gaps[cite: 8].
*   **Output Goal:** Generate counter-messaging narratives for sales teams to handle competitive objections[cite: 8].

## 3. InfoSec and Data Privacy Guardrails
*   **Free Tiers:** Never input proprietary product strategy, sensitive PII, or internal roadmaps into a free consumer model. These tools explicitly log inputs for future training[cite: 9]. Use free tiers strictly for public domain research[cite: 9].
*   **Enterprise Tiers:** Rely on enterprise-grade paid tiers that offer explicit zero-retention data privacy guarantees and SOC 2/GDPR compliance to protect competitive assets[cite: 8, 9].
